# 🏺 Dendê Softhouse - Time Rios

Este repositório contém os projetos desenvolvidos pelo time **Rios** durante o programa de trainee **Dendezeiro** da Dendê Softhouse. O objetivo central é a construção de um modelo de recomendação assertivo e personalizado para o aplicativo **Dendê Eventos**.

---

## 🚀 Desafios Implementados

### 1. Análise Exploratória de Dados (OAT 1)
Implementação de uma biblioteca de métricas estatísticas utilizando apenas **recursos nativos do Python** (sem o uso de bibliotecas externas como `numpy` ou `pandas`).

* **Principais funcionalidades:**
    * Cálculos de tendência central: média, mediana e moda.
    * Cálculos de dispersão: variância e desvio padrão.
    * Análise de relação: covariância.
    * Distribuição: frequências (absoluta, relativa e acumulada) e quartis.
* **Estrutura de dados:** Processamento de datasets representados como dicionários de listas.

### 2. Pré-processamento de Dados (OAT 2)
Desenvolvimento de algoritmos para transformar dados brutos em formatos consistentes para modelos de aprendizado de máquina, mantendo a premissa de utilizar apenas recursos nativos.

* **Tratamento de ruídos:** * Identificação e remoção de dados duplicados.
    * Tratamento de dados ausentes (NaN) por meio de imputação (média/mediana).
* **Transformação de variáveis:** * Implementação de técnicas de normalização e padronização (*Scalers*).
    * Codificação de variáveis categóricas (*Label* e *One-Hot Encoding*).
